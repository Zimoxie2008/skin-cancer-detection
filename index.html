Skin Cancer Detection Project

Your Name, Grade 12 Science Fair

Project Overview

This project investigates how image augmentation and optimization affect AI models in skin cancer detection. We aim to improve accuracy using various techniques on a public dataset.

Aim

To explore how augmentation and optimization improve skin cancer detection in images.

Method

Train models on augmented data, compare results, and fine-tune with learning rate schedulers and mixed precision.

Variables

Independent: Augmentation type, optimization settings
Dependent: Model accuracy
Control: Dataset, architecture
