Aim

The goal of this project is to develop an AI system capable of detecting skin cancer from images of pigmented skin lesions. This system will be trained using different techniques, specifically focusing on how image augmentation affects model accuracy and how model optimization can improve performance.

Part 1: Effect of Augmentation

Aim

To investigate how different image augmentation techniques influence the accuracy of the model in detecting skin cancer.

Method

The model will be trained on images with various augmentation techniques, including rotation, flipping, and color adjustments. Each set of augmented images will be used to see how the model responds and performs on unseen data.

Variables

Independent Variable: Augmentation techniques applied to the training data (e.g., rotation, flipping, color variations).
Dependent Variable: Model accuracy (e.g., AUC, loss).
Control Variables: Model architecture, learning rate, dataset.

Part 2: Model Optimization

Aim

To investigate how different model optimization techniques (e.g., learning rate schedules, weight decay) affect the performance of the model in detecting skin cancer.

Method

The model will be optimized using different techniques, such as learning rate scheduling and weight decay. The impact of these optimizations will be measured by comparing performance on a validation set after each change.

Variables

Independent Variable: Optimization techniques (e.g., learning rate scheduler, weight decay).
Dependent Variable: Model performance metrics (e.g., accuracy, AUC, loss).
Control Variables: Augmentation, model architecture, dataset.
